sNeatBookmarks is a clean and simple way to access your bookmarks through a neat tree view. It brings back the classic bookmark sidebar feel in a fast, modern popup that stays out of your way.

## Key Features

*   **Tree View Navigation**: Browse all your bookmarks and folders in a clear, nested list.
*   **Quick Search**: Find any bookmark instantly by typing a few letters in the search bar.
*   **Theme Presets**: Choose from several beautiful styles like Neon, Vintage, Dark, or Modern Light.
*   **Context Menus**: Right-click any bookmark to edit, delete, or open it in a new window.
*   **Custom CSS**: Full support for your own styles if you want to create a truly unique look.

## Quick Start

### Install from Chrome Web Store
1. Visit the [Chrome Web Store](https://chrome.google.com/webstore) (Coming Soon).
2. Click **Add to Chrome**.

### Manual Installation
1. Download this repository as a ZIP and extract it.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the extension folder.

## Overview

This extension is a modernized fork of the original Neat Bookmarks, updated to Manifest V3. It uses a high-performance tree rendering system and Chrome's native bookmark API to ensure your data is always in sync and loads instantly.

## FAQ

**Q: What does the 's' in the name mean?**
A: It stands for simple and lightweight application with no bloat, tracking, or ads.

**Q: Can I change how the popup looks?**
A: Yes, there are multiple theme presets in the settings, and you can even add your own custom CSS.

**Q: Does it sync with my Google account?**
A: Yes, it uses your native Chrome bookmarks, so if you use Chrome Sync, your bookmarks will be there.

**Q: Can I open multiple bookmarks at once?**
A: You can right-click a folder to open all bookmarks inside it in a new window or tabs.

**Q: Does it remember my last position?**
A: Yes, the extension can remember which folders you had open and where you were scrolling.
