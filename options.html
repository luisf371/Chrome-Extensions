Usage Tips

Undo latest closed tab

HOW: Check the hotkey option in Settings, then use the configured shortcut (Alt+Z).

Keyboard navigation in popup

HOW: Open the popup and use arrow keys to navigate.

CONTROLS: ←, → = Previous, Next page; ↑, ↓ = Select items; Enter = Restore.

Assign browser shortcut

HOW: Go to Extensions settings and click "Keyboard shortcuts" at the bottom.
